# インドネシア語

## 文構造の分類
インドネシア語の文構造は「主語-述語」。
但し、会話文では「述語-主語」になることもある。

- 平叙文
  - 名詞文: 述語が名詞である語。
    否定するには、 `bukan` を名詞の前に入れる。
    これは作文1のクラスです。
    > Ini kelas Menulis 1(satu).
    これは作文1のクラス**ではありません**。
    > Ini **bukan** kelas Menulis 1(satu).
  - 非・名詞文: 述語が名詞でない語。
- 疑問文
  - 名詞文: 述語が名詞である語。
    返答は**ya(はい)** か **Bukan(いいえ)** で答える。
  - 非・名詞文: 述語が名詞でない語。
