# indicative pronouns -fr

- この、その、あの
    
    ce(cet), cette
    
    ces, ces
    
    - 近遠を明確にしたい時
        
        ce [nom]-ci
        
        この〜
        
        ce [nom]-là
        
        あの〜
        
- これ、それ、あれ
    
    ### 弱形これ、あれ(c'est~でのみ使われる)
    
    ce, ce
    
    ce, ce
    
    ### 強形これ(cela, çaとの対比でのみ使う)
    
    ceci, ceci
    
    ceci, ceci
    
    ### 強形あれ
    
    cela(ça), cela(ça)
    
    cela(ça), cela(ça)
