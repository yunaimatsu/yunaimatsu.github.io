# 用言
## 作り方
1. `-다` を取って語幹を用意する
2. 語幹を変化させる
    変化のさせ方は3通り
    a. 変化しない
    b. `-으` がつく
    c. `-아` または `-어` をつける
    ※つけたあと、母音がくっつく
    - 만나다 -> 만나아- -> 만나-
    - 서다 -> 서어- -> 서-
    - 펴다 -> 펴어- -> 펴-
3. 助詞や接続詞をつける
