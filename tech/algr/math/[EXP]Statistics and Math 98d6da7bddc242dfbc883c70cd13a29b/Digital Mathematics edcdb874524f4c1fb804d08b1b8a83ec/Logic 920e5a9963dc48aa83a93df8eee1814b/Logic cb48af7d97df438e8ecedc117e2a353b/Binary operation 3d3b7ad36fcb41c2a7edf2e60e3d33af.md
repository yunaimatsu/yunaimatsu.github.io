# Binary operation

# Logic chart

**If “A and B” is true**

| A | B |  |
| --- | --- | --- |
| true | true | true |
| true | false | false |
| false | true | false |
| false | false | false |

**A or B**

| A | B |  |
| --- | --- | --- |
| true | true |  |
| true | false |  |
| false | true |  |
| false | false |  |

**A xor B**

| A | B |  |
| --- | --- | --- |
| true | true |  |
| true | false |  |
| false | true |  |
| false | false |  |
