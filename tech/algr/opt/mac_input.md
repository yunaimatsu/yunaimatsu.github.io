# 英語だけで多言語入力をすることのメリットとデメリット
### メリット
- パソコンの動作が軽くなる。
- 多言語学習者にとっては言語切り替えの手間を省けるため楽。
### デメリット
- 予測変換が使えない。

# 配列
`option`（以外`opt`）と何かの文字キーを同時に入力すると、特殊記号が打てる。
