# Inverse Trigonometric Functions　(Cyclometric Functions)

$$
y=\sin^{-1} x
\\
y=arc\sin x
\\
y=\cos^{-1} x
\\
y=arc\cos x
$$

- arcsin
- arccos
- arctan
