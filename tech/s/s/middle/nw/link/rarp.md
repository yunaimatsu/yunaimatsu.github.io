# RARP(reverse_adress_resolution_protocol)
IP = RARP(MAC)
MAC_adressからIP_adressを知りたい時に使う。

- 普通のComputer
    
    Asign IP_adress with DHCP server or server_computerなら手動で入力
    
- Built-in computer
    
    Asign IP-adress_input_interaceがなかったりDHCPに対応していなかったりする。
    

RARP serverを用意。
