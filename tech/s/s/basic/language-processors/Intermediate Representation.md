# Intermediate Representation

- 中間コードの生成
    
    ソースコードが解析された後、コンパイラは中間コードを生成します。中間コードは、プログラムを機械語に変換する前の中間段階であり、機械語と似た形式で表現されます。このステップは、プログラムを異なるプラットフォームで実行できるようにするために、必要な場合があります。
