# .md(mark_down)

# features

- lightweight
- plain-text formatting syntax
- easy to read and writ
- can be converted into HTML and other formats

# syntax

### heading

```bash
# Heading 1
## Heading 2
### Heading 3
```

### **Emphasis (Bold and Italics):**

```bash
**Bold text**
*Italic text*
```

### list

```bash
- Item 1
- Item 2
  - Subitem A
  - Subitem B
```
